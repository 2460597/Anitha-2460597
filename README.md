# Prime Server

Simple Java HTTP server that finds prime numbers up to a given number.

How to run locally:

Prerequisites: Java 11+ and Maven installed.

1. Build:

   mvn package

2. Run (option A - with Maven):

   mvn exec:java

   or (option B - run the jar):

   java -jar target/prime-server-1.0-SNAPSHOT.jar

Open your browser at http://localhost:8080 and use the form.
